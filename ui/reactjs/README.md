# 🛡️ Healthcare Claims Fraud Detection Dashboard

A modern **React + TypeScript** analytics dashboard for Healthcare Claims Fraud Detection built on top of a **FastAPI** backend. The application provides interactive dashboards, KPI monitoring, fraud analytics, anomaly detection, machine learning model evaluation, and executive summaries for healthcare claims.

---

# Overview

The dashboard enables healthcare organizations, insurance providers, and fraud investigation teams to visualize fraudulent claims, monitor fraud trends, identify suspicious providers, analyze root causes, evaluate ML model performance, and generate executive insights.

The frontend communicates with a FastAPI Fraud Detection API and renders real-time visualizations using reusable React components.

# Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd project
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm run dev
```

Production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
